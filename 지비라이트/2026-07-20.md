## 2026-07-20 업무 기록

### 프로젝트

- 지비라이트 제조표준서 관리 기능 개발

### 수행 작업

- 제조 기준 데이터를 선택해 화면에 반영하고 저장하는 흐름을 점검했다.
- 여러 상세 데이터 묶음을 화면 표시용과 저장용 원본 데이터에 일관되게 반영하도록 데이터 흐름을 정리했다.
- 신규 작성 및 수정 중인 문서를 우선 조회하는 목록/상세 조회 구조를 설계했다.
- 기본정보, 공정 정보, 파트 정보, 상세 데이터, 설비 기준 데이터를 포함하는 다중 결과셋 조회를 구성했다.
- 조회 결과를 화면 입력값과 그리드 원본에 복원하는 바인딩 함수를 구현했다.

### 기술 및 설계

- SQL Server의 TEMP 우선 조회 패턴을 사용해 작성 중 데이터와 확정 데이터를 구분했다.
- 부모 데이터와 상세 데이터의 연결키 및 조회 순서를 정리했다.
- C# DataTable 기반 그리드 바인딩과 저장 원본 동기화 방식을 적용했다.
- 팝업 반환 데이터에 숨김 식별값을 포함해 후속 저장 시에도 관계를 유지하도록 개선했다.

### 검증 및 다음 작업

- 작업 기준 데이터의 팝업 선택 및 TEMP 저장 흐름을 확인했다.
- 일부 그리드에서 상세 컬럼 표시가 누락되는 현상을 확인했으며, 다음 작업에서 조회 결과와 화면 컬럼 매핑을 대조해 해결할 예정이다.
